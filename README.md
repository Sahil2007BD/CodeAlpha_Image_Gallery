# 📸 Image Gallery Project

A modern, responsive **Image Gallery Web App** built using **HTML, CSS, and JavaScript**. This project features smooth animations, interactive navigation, and a clean UI — perfect for showcasing images in a stylish way.

---

## 🚀 Features

✨ **Responsive Layout**

* Works seamlessly on desktop, tablet, and mobile devices
* Grid-based design using CSS Flexbox/Grid

🎯 **Interactive Navigation**

* Next / Previous buttons for easy browsing
* Keyboard navigation support (optional enhancement)

🔍 **Lightbox View**

* Click any image to open in fullscreen modal
* Smooth transitions and overlay effects

🎨 **Hover Effects & Animations**

* Image zoom / overlay effects on hover
* Smooth CSS transitions for a premium feel

🗂️ **Category Filters (Bonus Feature)**

* Filter images by category (e.g., Nature, City, People)
* Dynamic UI updates using JavaScript

🌈 **Image Effects (Bonus Feature)**

* CSS filters like grayscale, blur, brightness
* Toggle effects dynamically

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling, layout, animations
* **JavaScript (Vanilla)** – Interactivity and logic

## 📸 How It Works

1. Gallery displays images in a responsive grid
2. Clicking an image opens a lightbox modal
3. Users can:

   * Navigate using Next/Prev buttons
   * Close the modal
4. Filters allow switching between categories
5. Hovering adds visual effects
